# projet-contrôle-de-version
Article : Sécurité de Git
Ce projet est un site web très simple (HTML/CSS) qui permet de lire un article sur les bonnes pratiques de sécurité à adopter avec Git.
#Comment l'utiliser
Il n'y a rien à installer ! Pour consulter l'article sur votre ordinateur :

Clonez ce dépôt :

Bash
git clone https://github.com/[ton-nom-utilisateur]/[nom-du-repo].git
Ouvrez le dossier du projet.

Double-cliquez sur le fichier index.html pour l'ouvrir dans votre navigateur.
